# src/scenes

Cenas e cenarios oceanicos (zonas de profundidade, fundo do mar, paredes de rocha, raios de sol, etc).

- `rock-walls.js` — paredoes laterais de rochas (pedras irregulares geradas proceduralmente nas bordas do mundo).
- `sun-rays.js` — raios de sun subtis e desfocados proximos a superficie (god rays), com fade conforme a profundidade.
