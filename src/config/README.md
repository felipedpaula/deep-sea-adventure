# src/config

Configuracoes e dados do jogo (modulos JS ou JSON): zonas, conquistas, itens, tema, etc.

- `theme.js` — valores de tema (cores, espaçamentos, raios) reutilizaveis entre CSS e JS.
- `canvas.js` — configuracoes do Canvas (cor de fundo inicial, clamp de DPI maximo).
- `game.js` — configuracoes do loop (clamp de delta time, fps alvo informativo).
- `submarine.js` — configuracoes do submarino (dimensoes, velocidade base, multiplicador de aceleracao, suavizacao, inclinacao maxima, cores).
- `world.js` — configuracoes do mundo (dimensoes, escala pixels/metro, profundidade inicial, gradiente de fundo por profundidade, paredes de rocha).
